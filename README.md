# TaskFlow

기업용 태스크 관리 시스템

### 개발기간
2025.06.13 - 2025.06.19

### 깃허브 링크
https://github.com/ParkJin0814/taskFlow

---

## 기술스택
- Java 17
- Spring Boot 3.4.6
- Spring Data JPA
- Spring Security
- MySQL
- Gradle 8.14(wrapper)

## 주요 요구사항
- 사용자 권한에 따른 접근 제어
- 태스크의 CRUD 및 상태 관리
- 댓글을 통한 협업 기능
- 대시보드용 통계 데이터 제공측
- AOP 기반의 상세 활동 로깅

## 기능 설명
- 회원 관리
    - 회원가입: 이메일 형식 ID, Bcrypt 암호화된 비밀번호, 권한 부여(일반 USER/관리자 ADMIN)
    - 로그인: JWT 기반 인증
    - 회원 탈퇴
- Task 관리
  - 태스크 생성, 수정, 삭제(Soft Delete)
  - 속성: 제목, 설명, 우선순위(LOW, MEDIUM, HIGH), 담당자, 생성자, 마감일, 상태(TODO, IN_PROGRESS, DONE), 생성일, 수정일 등
  - 페이징, 검색 기능
- 댓글 관리
  - 댓글 생성, 수정, 삭제(Soft Delete)
  - 태스크별 댓글 조회 및 검색
- 대시보드
  - 전체 태스크 통계 (총 개수, 상태별 개수, 완료율)
  - 내 태스크 요약
- 활동 로그
  - 모든 데이터 변경 이력 기록
  - 로그 저장 및 조회
